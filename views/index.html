API Basejump: Request Header Parser Microservice

User Story: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.

Example creation usage:

https://agate-heart.glitch.me/new/https://www.google.com

Example creation output:

{ "original_url":"https://www.google.com", "short_url":"https://agate-heart.glitch.me/8170" }

Usage:

https://agate-heart.glitch.me/8170

Will redirect to:

https://www.google.com
